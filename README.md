# DeliveryGo

쉽고 빠른 택배조회 서비스

-----------------
<h3>프로젝트 소개</h3>
모든 쇼핑몰의 배송 정보를 한 눈에 파악가능.
휴대번호로 한 번 저장한 배송 정보는 언제든지 확인 가능.

-------------

<li>2023.3 ~ 2023.4.14</li>
